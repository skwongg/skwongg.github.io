    The #map method

    February 6, 2015

    Enumerable #map Method!



    The map method takes an array and runs a block of code for each element in the object. An array is an object that stores other objects in a numerically indexed fashion. An example of an array would be ArrayExample = [1,2,3]. This example shows an array named "ArrayExample" which contains 3 elements in it. If we wanted to access each element, we would call them by their position. Array positions are indexed starting from zero, the Zero-th position is the first element of an array. In ArrayExample, if we wanted to pull "1" out, we would use: ArrayExample[0]. If we wanted to pull "2" out, we would use ArrayExample[1] and finally if we wanted to pull "3" out we would use ArrayExample[2].

    The map method takes an array and applies a block of code to each individual element in the array.

    Example

    nums = [1,2,3]

    nums.map { |x| x * x }

    output =>[1,4,9] in a new array.

    In the example above, the |x| defines the object being modified and the x*x represents the method being applied to each of the elements in the array. The {} represent the confines of the code block and represent where the code block starts and where it ends.

    There is also a destructive map! method in which modifies the original array.

    Example

    nums = [1,2,3]

    nums.map! { |x| x * x }

    output =>[1,4,9] which modifies the original array.

    The map method is kind of like an assembly line for code. The array being passed through the method are the parts being assembled. The final product comes out as the output which has been modified by the code block of the map method. Thanks for taking the time to read my blawg. :)
